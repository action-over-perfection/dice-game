## 주사위 게임
* 플레이어와 컴퓨터가 각각 주사위를 굴린다.
* 주사위는 1~6까지의 숫자가 나온다.
* 더 큰 숫자가 나온 쪽이 승리이고, 같은 숫자면 무승부이다.
* 5판을 진행하고 최종 승패를 출력한다.

### 입력
* 플레이어 이름 입력
```
플레이어 이름을 입력해 주세요.
철수
```
* 라운드마다 엔터 입력으로 주사위 굴리기
```
Enter를 눌러 주사위를 굴려주세요.
```

### 출력
* 게임 시작 메시지
```
=== 주사위 게임 ===
5라운드 대결입니다.
```
* 라운드 결과
```
[라운드 1]
철수: 3
컴퓨터: 5
컴퓨터 승리!
```
* 최종 결과
```
=== 최종 결과 ===
철수: 2승 1무 2패
컴퓨터: 2승 1무 2패
무승부입니다!
```
